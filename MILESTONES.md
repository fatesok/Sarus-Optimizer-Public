## Current public milestone

The current public milestone is **M3 - Toronto stadium benchmark summary**.

Completion criteria:

- benchmark scenario name is clear
- assumptions are documented at a public-safe summary level
- outputs have been reviewed
- units and limitations are stated
- no source code, proprietary data, or internal tasks are exposed
- sensitivity results are included where needed

## Evidence milestones

The most important credibility milestones are not code releases. They are evidence releases:

1. a clear benchmark methodology
2. a reviewed summary of results
3. sensitivity results showing how assumptions affect recommendations
4. selected visuals that explain tradeoffs
5. limitations stated plainly

## What should remain private

The following should not be treated as public milestones unless explicitly approved later:

- source-code release
- private algorithm release
- full task-board release
- raw data release
- unrevised experimental results
- internal bug or limitation logs
- exact calibration details that create competitive advantage
