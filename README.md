# Sarus-Optimizer Public Information Pack

This repository contains public-facing material for Sarus-Optimizer.

It is designed for gradual public communication before any source code is released. The documents explain the project vision, public roadmap, milestone direction, use cases, and benchmark-summary approach without exposing implementation internals.

## Public documents

- [Vision](VISION.md)
- [Roadmap](ROADMAP.md)
- [Milestones](MILESTONES.md)
- [Use Cases](USE_CASES.md)
- [Benchmark Summaries](BENCHMARK_SUMMARIES.md)

## Publicization rule

Public material should describe what Sarus-Optimizer is trying to solve, why it matters, and how progress will be evaluated.

Public material should not disclose:

- source code
- private algorithms or implementation details
- internal task lists
- unresolved bug lists
- proprietary data
- exact private calibration assumptions
- unreviewed numerical claims

## Suggested publishing sequence

1. Publish the vision statement.
2. Publish the public roadmap.
3. Publish milestone status.
4. Publish use cases with plain-language examples.
5. Publish benchmark summaries only after the assumptions and results have been reviewed.

## Current public status

Sarus-Optimizer is in private development. Public communication should position it as a planning-optimization project focused on explainable facility, transport, capacity, and investment decisions.
